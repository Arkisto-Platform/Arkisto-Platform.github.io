Tools - Data Capture and export

There are a wide variety of tools for exporting data into the standards-based Arkisto world from a variety of data sources so that data can be stored in Standards-based sustainable format.

Repository export to RO-Crate tools #

Omeka #

Status: Working scripts that need updating to the latest version of RO-Crate.

This git repository has scripts for exporting data from the widely used Omeka Classic and Omeka S repsitory/CMS tools into RO-Crate format. The reference to "datacrate" is to a previous version of the standard.

Heurist #

Status: Work in progress

This git repository houses code that exports Heurist databased into RO-Crate.
