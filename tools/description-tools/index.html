Describo #

Status: Release candidate

Describo is a domain-configurable cross-platform desktop tool for describing and packaging data using the [RO-Crate] standard.

Descibo can package:

  • Any research data for deposit in a general-purpose repository (such as the UTS data repository).

Describo has been engineered with potential new functionality in mind:

  • The ability to connect to multiple cloud storage platforms (Dropbox, Microsoft OneDrive, Google Drive etc)

  • The ability to run as a web application so it can be used from any device (while talking to various platforms).

  • The ability to function as a collection-development tool

XLRO #

Status: Beta

XLRO is a tool for describing and packaging data using spreadsheets. It is a commandline tool which can be used by eResearch and data management staff to create Excel templates that can be filled out by non-technical users.
