Standards - Archive and Package (arcp) Identifiers

In order to make Arkisto repositories interoperable so that data can be federated, aggregated and moved between providers we are exploring the use of the The Archive and Package (arcp) URI scheme; ARCP identifiers allow local data stores to carve out a namespace for their data, in a similar way to DNS, so that local identifiers can be minted and made unique.

Sustainability: This will allow OCFL repository content to be merged without clashes - making it possible for legacy content to be re-housed in larger repositories if needed - and enable distributed repository management.
